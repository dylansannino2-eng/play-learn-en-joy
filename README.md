# Learn English With Games

An interactive platform to learn English through fun games and activities.

## Features

- **The Translator**: Translate Spanish phrases to English in real-time
- **Word Battle**: Compete to find words matching specific criteria
- **Movie Interpreter**: Learn from movie clips with subtitles
- **Multiplayer**: Play with friends in real-time

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`

## Technologies

- React + TypeScript
- Tailwind CSS
- Supabase (Backend & Real-time)
- Vite
